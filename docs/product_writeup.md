# Help U - Bookkeeper: Product & Technical Specification

## 📊 Overview
**Help U - Bookkeeper** is an AI-powered accounting and GST management solution designed for small and medium businesses (SMBs). It simplifies bookkeeping by allowing users to record transactions through **WhatsApp** (images, voice, text) and manage their business data via a **Web Portal**. The system integrates directly with **Google Drive and Google Sheets**, using them as a transparent and user-accessible "database" for financial records.

---

## 📸 WhatsApp Interface (The "AI Bookkeeper")
The WhatsApp bot acts as the primary data entry point, leveraging AI to extract information from unstructured inputs.

### 1. AI-Powered Recording
- **Image Extraction**: Users send photos of bills/receipts. The system uses **GPT Vision models** (e.g., GPT-4o-mini) to extract GSTINs, HSN codes, dates, amounts, and tax splits.
- **Voice Transcription**: Users send voice notes (e.g., *"Sold items for 500 to ABC Corp"*). The system uses **Faster-Whisper** for high-accuracy local transcription.
- **Natural Language Processing**: Extracted text (from voice or typed messages) is processed by AI to detect intent and extract transaction details.

### 2. Conversational Business Intelligence
- **Stats & Analysis**: Users can type "Stats" for monthly totals or "Analysis" for a deep business report (growth, risks, GST compliance) generated by an **AI Consultant Agent**.
- **On-Demand Advice**: Users can ask specific business questions (e.g., *"How can I reduce my tax liability?"*), which the AI answers using the context of their business data.

### 3. Automated Reporting & Document Generation
- **Instant Invoices**: Generates professional PDF invoices from ledger data on demand.
- **GSTR-1 Preparation**: Generates government-compliant **GSTR-1 JSON** files for the current month and delivers them directly via WhatsApp.
- **Cloud Archiving**: Automatically uploads all bill images to a structured "Help U" folder on the user's Google Drive.

---

## 🎨 Web Portal (The "Business Dashboard")
A React-based frontend for deeper analysis, configuration, and manual data management.

### Key Features:
- **Business Onboarding**: Multi-step flow for connecting Google accounts, setting up Drive folders, and configuring business profiles (GSTIN, legal name).
- **Interactive Dashboard**: Visualizes sales/purchase trends, tax liability, and payment status (Paid vs. Credit).
- **Transaction Management**: View, edit, and manually add transactions synced with the Google Sheet ledger.
- **Settings & Integrations**: Manage Google Drive access, invoice templates, and Stripe subscription status.

---

## ⚙️ Technical Architecture

### 1. Frontend (FE)
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4.x
- **Routing**: React Router 7
- **State**: React Context API
- **Deployment**: Nginx (Dockerized)

### 2. Backend (BE)
- **Framework**: FastAPI (Python 3.10+)
- **Database**: PostgreSQL (GCP Cloud SQL) for user metadata; Google Sheets for financial data.
- **ORM**: SQLAlchemy + Alembic for migrations.
- **Auth**: JWT-based session management + Google OAuth2 for Drive access.

### 3. AI & ML Stack
- **Vision/LLM**: OpenAI API (GPT-4o-mini / GPT-4o).
- **Speech-to-Text**: Faster-Whisper (optimized local inference).
- **Agentic Logic**: Custom `ConsultantAgent` for business analysis and advice.

### 4. Integrations
- **WhatsApp Business API**: Messaging and media handling.
- **Google API**: Drive (Storage) and Sheets (Ledger).
- **Stripe**: Subscription and payment processing.

---

## 🚀 Infrastructure & DevOps
- **Cloud Platform**: Google Cloud Platform (GCP).
- **Serverless**: Backend and Frontend hosted on **Cloud Run**.
- **Managed DB**: **Cloud SQL** (PostgreSQL).
- **CI/CD**: GitHub Actions for automated deployment.
- **Containerization**: Docker-based workflows for local dev and production parity.

---

## 🔒 Security & Data Privacy
- **OAuth2**: Secure, scoped access to user's Google Drive (only "Help U" related files).
- **Encrypted Storage**: Refresh tokens and business metadata stored in encrypted RDS.
- **Privacy**: Financial data resides in the user's own Google account, giving them full ownership and transparency.
