# Help U - Bookkeeper Project Guide

## Build & Setup
- Initialize virtual environment: `python -m venv venv`
- Activate virtual environment: `venv\Scripts\activate`
- Install dependencies: `pip install -r requirements.txt`

## Test
- Run tests: `pytest`

## Lint & Typecheck
- Lint: `pylint src`
- Typecheck: `mypy src`
